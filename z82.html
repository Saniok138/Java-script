Zadanie 82: Model DOM

1. Co to jest model DOM?

Model DOM (Document Object Model) to sposób opisu złożonych dokumentów XML i HTML w postaci hierarchii modeli, gdzie niektóre obiekty są nadrzędne nad innymi, a cała strona tworzy drzewo obiektów modelu obiektowego.

Mechanizm działania DOM polega na tym, że kod HTML zamieniany jest przez przeglądarkę na stronę WWW, a przeglądarka przechowuje informacje o stronie w postaci modelu obiektowego, który ma hierarchię obiektów. Udostępniane są metody oraz właściwości obiektów.

2. Przykład kodu + schemat/drzewo DOM

Przykładowy kod HTML:

<!DOCTYPE HTML>
<html>
 <head>
 <title>Obiekty</title>
 <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
 </head>
 <body>
 <p>Strona DOM<b>pokaz</b></p>
 </body>
</html>
